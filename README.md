# 🌊 FloatChat

AI-Powered Conversational Interface for ARGO Ocean Data Discovery and Visualization

## 📖 About

FloatChat is an AI-powered conversational interface designed to democratize access to global oceanographic data from ARGO floats. Built with React, Three.js, and TailwindCSS, it provides an immersive ocean-themed experience for exploring ocean data.

**Mission**: Democratizing access to global oceanographic data through AI + Visualization.

## ✨ Features

- 🌊 Ocean Background - Three.js animated ocean with wave effects
- 🖱️ Interactive Cursor - Ripple effects and dynamic glow
- 🤖 AI-Powered Chatbot - Natural language queries for ocean data
- 📊 Data Visualization - Interactive dashboards and maps
- 🎨 Futuristic UI - Glassmorphism cards with gradient effects
- 📱 Responsive - Works on all screen sizes

## 🛠️ Tech Stack

- React 18.x - UI Framework
- Three.js 0.158.x - 3D Rendering
- Framer Motion 10.x - Animations
- TailwindCSS 3.x - Styling
- Vite 4.x - Build Tool
